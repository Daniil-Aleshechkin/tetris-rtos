Detailed Description

Structures and functions for managing the state of a Tetris game.

This header file defines the data structures that represent the state of a Tetris game, including the game board, the current piece in play, the piece held in reserve, and the queue of upcoming pieces. It also declares functions for initializing the game state, determining the starting position for new pieces, and cleaning up game state resources.
