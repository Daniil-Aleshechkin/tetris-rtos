Detailed Description

Command Line Interface (CLI) for Tetris game running on FreeRTOS.

This header file defines the interfaces for a Command Line Interface (CLI) that allows users to interact with the Tetris game and controls the settings via a serial terminal. It includes definitions for initializing the CLI, processing user input, and executing commands.
