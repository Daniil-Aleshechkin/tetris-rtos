Detailed Description

Defines the rotation states for the Tetris "I" piece.

This header file contains the definitions of the Tetris "I" piece's layout for each of its four possible rotations (0, 90, 180, and 270 degrees). Additionally, it declares a function for retrieving the "I" piece's layout based on its current rotation.
