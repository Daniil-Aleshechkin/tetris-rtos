Detailed Description

Defines the rotation states for the Tetris "S" piece.

This header file contains the definitions for the layout of the Tetris "S" piece for each of its four possible rotations (0, 90, 180, and 270 degrees). It also declares a function for retrieving the "S" piece's layout based on its current rotation, facilitating its placement and movement on the game board.
