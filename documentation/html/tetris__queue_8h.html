Detailed Description

Functions and structures for managing the Tetris piece queue.

This header file provides the definitions for managing a queue of Tetris pieces, including generating a bag of pieces, adding it to the queue, and retrieving pieces from the queue. The queue is used to determine the order in which Tetris pieces appear during gameplay.
