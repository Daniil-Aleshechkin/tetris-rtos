\doxysubsection{Detailed Description}
Functions for Tetris piece operations and game state management. 

This header file provides declarations for functions involved in rotating, moving, dropping Tetris pieces, and managing the Tetris game state. It includes utilities for printing the game state to a terminal or a specific display and for resetting the game state.
