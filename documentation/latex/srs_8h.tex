\doxysubsection{Detailed Description}
Defines the Super Rotation System (SRS) offsets and provides functions for Tetris piece rotations. 

This header file includes the definitions of the SRS offsets for rotating Tetris pieces according to the Super Rotation System (SRS). It contains arrays of offsets for each rotation state and functions to retrieve these offsets depending on the piece type and its current rotation. Each offset array corresponds to the priority with which each offset is applied during a rotation attempt. The system checks for each offset in order to find a valid piece placement (also known as a \char`\"{}kick\char`\"{}). The first offset that allows for a valid placement will be applied to perform the rotation.

Retrieves SRS offsets based on piece type and rotation with priority consideration. 

Determines the appropriate SRS offsets to use when rotating a piece from its current orientation to a new orientation, based on the piece type, the intended rotation direction, and the priority of offset application. The first offset that allows for a valid placement (kick) will be applied to perform the rotation.
