\doxysubsection{Detailed Description}
Defines the static state for the Tetris \char`\"{}\+O\char`\"{} piece. 

This header file contains the definition for the layout of the Tetris \char`\"{}\+O\char`\"{} piece. The \char`\"{}\+O\char`\"{} piece is unique in that it does not change its orientation with rotation. The layout is a 2x2 square, which remains constant regardless of rotation commands.
